Welcome to KBWC API Client's documentation!
===========================================

:mod:`kbwc_api_client` is a client library for working with the API to the
`WorldCat knowledge base <http://oclc.org/knowledgebase>`.  It provides two
possible clients:

 :mod:`OpenURL`
   A client for the OpenURL endpoint
 :mod:`Rest`
   A client for the REST / A to Z part of the API
